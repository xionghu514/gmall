# 谷粒商城

> 尚硅谷 谷粒商城是一套电商项目, 包括前台商城系统以及后台管理系统, 基于 SpringCloud + SpringCloudAlibaba + MyBatis-Plus实现, 前台商城系统包括：用户登录、注册、商品搜索、商品详情、购物车、下订单流程、秒杀活动等模块。后台管理系统包括：系统管理、商品系统、优惠营销、库存系统、订单系统、用户系统、内容管理等七大模块。
